# AI Tools & Workflow – DelhiGlow
### SuperXgen AI Startup Buildathon 2026 Submission

---

## What We Built
**DelhiGlow** is a city-based luxury bridal salon marketplace for Delhi NCR.
Users can browse salons, filter by area and service type, view full details, and book appointments — all in one seamless experience.

---

## AI Tools Used

### 1. Claude (Anthropic)
**Used for:** Architecture planning, all code generation, salon data creation, UI copy, and debugging.

- Generated the full React frontend (5 pages + 3 components)
- Built the Node.js/Express backend with filtering, search & booking API
- Created realistic salon dummy data (6 salons, 36 services with real Delhi pricing)
- Wrote all UI copy — hero text, taglines, how-it-works section, footer
- Debugged routing, form validation and API response handling

### 2. Vercel
**Used for:** One-click frontend deployment with automatic HTTPS and CDN.

### 3. Railway / Render
**Used for:** Backend Node.js API deployment.

### 4. Figma (AI features)
**Used for:** Initial wireframe exploration and color palette validation before coding.

## Development Workflow

```
Idea → Claude (architecture) → Claude (code) → Local test → Vercel deploy → Demo
```

Total build time: ~20 hours (within 48hr hackathon window)

**Code written by hand:** ~5%
**Code generated/assisted by AI:** ~95%

---

## Key AI Innovation Points

1. **Entire dummy dataset generated by Claude** — realistic Delhi salon names, locations, pricing (₹400–₹40,000), service durations, specialities, and contact numbers that feel authentic.

2. **Design system prompted via Claude** — the crimson + gold luxury palette, typography choices (Playfair Display + Inter), and component hierarchy were all designed through prompting.

3. **Zero boilerplate time** — no time wasted on setup. Claude generated the full project structure, package.json files, and wiring in one go.

4. **Iterative refinement** — filters, booking flow, and success page were all refined through follow-up prompts without rewriting from scratch.

## What Makes DelhiGlow a Real Startup Idea

- Delhi's wedding industry is worth ₹1.5 lakh crore annually
- Bridal beauty bookings are high-intent, high-value (avg ₹8,000–₹40,000)
- No dominant city-specific platform exists — gap in the market
- Monetization: Commission per booking (15–20%) + Featured salon listings
